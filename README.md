# Patchwork Assurance

*Working name. Actively in development.*

An AI-native tool that helps businesses understand how the fast-moving, state-by-state patchwork of
AI-regulation law applies to their specific situation. It grounds its answers in the actual statutory
text and is designed so that new jurisdictions, court decisions, and federal action can be folded in as
the landscape shifts.

**This README is intentionally minimal while the project is being built.** The current, controlling
plan, architecture, and roadmap live in [`docs/ROADMAP.md`](docs/ROADMAP.md). A full README will be
written once the application is complete.

## Not legal advice

This is an educational / portfolio tool, not a compliance product and not legal advice. The relevant
laws are new, unlitigated, and subject to change. Consult a licensed attorney for actual compliance
decisions.
